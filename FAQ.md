# Tectonic Engine 2 - FAQ

## 1. Is the code legal? Why do some function names seem similar to GoldSrc/Source?

The codebase for Tectonic Engine 2 was built entirely from scratch. While it has naming conventions similar to engines like GoldSrc or Source, no code has been copied from those engines.

Development primarily referenced the Pathos Engine (MIT) as a learning resource, but all implementations are original.

## 2. I see it uses the VBSP format. Is this legal?

Yes. VBSP is simply a file format, similar to formats like PNG or JPEG. It is not patented or restricted.

The code used to load and parse VBSP files in Tectonic Engine 2 was written from scratch, with reference to:
https://github.com/tasbox-org/source-parsers/
